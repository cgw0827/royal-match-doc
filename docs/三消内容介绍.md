

## 消除游戏介绍

### 道具消除时障碍物和草的规则
|          | 草                             | 障碍物                           | 普通道具 |
| -------- | ------------------------------ | -------------------------------- | -------- |
| 普通消除 | 下方                           | 周围                             | 无       |
| 导弹     | 消除路径中的草                 | 是                               |          |
| 螺旋桨   | 消除周围四个道具，并飞向一个草 | 消除周围四个道具，并飞向一个障碍 |          |
| 火药桶   | 是                             | 是                               | 是       |
| 灯球     |                                | 普通道具爆炸后消除障碍           | 是       |
