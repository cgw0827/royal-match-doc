
## 三消游戏模版   

1. 一款当下最流行的及时消除三消游戏，核心玩法和关卡配置复刻 RolyalMatch。 
2. 游戏包大小20MB
3. 目前实现200关

### 版权说明
1. 购买后代码可以商用。
2. 美术资源和音频资源来源于网络，仅用于实现游戏功能，不可商用。

### 使用的库
1. [跨多平台的sdk框架](https://gitee.com/lecoolgamesdk/sdk)
